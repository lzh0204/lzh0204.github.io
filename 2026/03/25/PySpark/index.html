一、简单介绍一下

Spark 是一款专门用来处理超大数据的分布式计算引擎，简单来说，普通电脑算不动的大数据，交给Spark就能很快算完

PySpark 就是用Python语言来写大数据处理程序的工具

PySpark 的编程模型是数据输入、数据计算和数据输出三大步骤。首先要通过 SparkContext 完成数据读取，然后把读取到的数据转换为 RDD 对象，调用 RDD 的成员方法完成计算，最后调用 RDD 的数据输出相关成员方法，将结果输出到 list、元组、字典、文本文件、数据库等

二、数据输入

PySpark 支持多种数据输入，在数据输入完之后会得到一个 RDD 对象（弹性分布式数据集），然后以 RDD 对象为载体对数据进行处理

1、Python数据容器转RDD对象

支持转换的数据容器有list，tuple，set，dict，str，其中str容器会输出单个字符，dict 容器会输出它的所有 key 值

3、reduceByKey算子

reduceByKey 算子是 Spark 中最核心、最常见的键值对聚合算子，专门用于相同的Key的Value进行聚合计算

作用：

  • 只处理键值对格式的数据
  • 自动按 Key 分组
  • 对相同 Key 的所有 Value 执行定义的聚合逻辑
  • 按照K-V结构输出

注意：K-V型的 RDD 就是二元元组 [(‘a’,1),(‘b’,2)]，其中每个元组的第一个值是 Key ，第二个为 Value

4、filter算子

filter 是一个转换算子——筛选、清洗、数据过滤等

作用：

  • 遍历 RDD 中的每一条数据
  • 根据写入的布尔条件判断：true 保留，false 丢弃
  • 返回一个新的 RDD ，不修改原数据

5、distinct算子

distinct 是一个去重算子，去掉重复元素，只保留唯一值

作用：

  • 对 RDD 里的所有数据全局去重

  • 重复数据只保留一份

  • 返回一个新的 RDD

  • 支持普通类型、字符串、元组、对象的数据容器

  • 支持升序/降序和复杂对象排序（元组、自定义对象等）
