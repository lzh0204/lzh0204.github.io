一、简介

NumPy 是Python中专门用来做数值计算、矩阵运算的核心库。Python中原本没有数组/矩阵，**NumPy** 给他补上了这套能力

NumPy 的作用：

  • 创建、操作多维数组：一维向量、二维矩阵、三维图片张量都能存
  • 快速数学运算：求和、均值、方差、矩阵乘法、逆矩阵、随机数等

二、Ndarray 对象

ndarray（N维数组对象）是用来存储同类型数据的多维表格，所有向量化计算、矩阵运算都围绕他展开

核心特点

  1. 固定类型：创建后，数组内所有的元素的数据类型（dtype）是相同的。这使其内存占用更紧凑，计算速度更快
  2. 多维性：可以是一维（向量）、二维（矩阵）、三维甚至更高维（张量）
  3. 向量化操作：可以直接对整个数组或某个维度进行数学运算，而无需编写显示的循环，这通常比原生Python列表快一到两个数量级
  4. 内存高效：数据存储在连续的内存块中，并利用底层 C 和 Fortran 库进行优化

创建方式

基本的 ndarray 是使用 NumPy 中的数组函数创建的

它从任何暴露数组接口的对象或从返回数组的任何方法创建一个 ndarray

参数 描述
object 任何暴露数组接口的方法的对象都会返回一个数组或任何（嵌套）序列
dytpe 数组的所需数据类型，可选
copy 可选，数组是否被复制，默认为True
order C（按行）、F（按列）、A（任意，默认）
subok 默认情况下，返回的数组被强制维基类数组，如果是True，则返回子类
ndmin 指定返回数组的最小维度

三、索引和切片

NumPy 数组的索引和切片比Python列表更强大，支持一维、多维、布尔、整数数组索引，是数据处理、机器学习的基础

一维数组

一维数组的索引和切片和Python列表用法几乎一致

  • 索引： arr[索引值] （索引从0开始）
  • 切片： arr[起始:结束:步长] （左闭右开，不包括结束位置）

二维数组

二维数组可以理解为矩阵，有两个维度：行（axis=0），列（axis=1）

语法： arr[行索引,列索引] 或 arr[行切片,列切片]

高级索引

除了普通索引切片， NumPy 还支持布尔索引、整数数组索引，筛选数据也很方便

  • 布尔索引（条件筛选）

  • 整数数组索引（自定义取元素）

传入一个索引数组，精准取多个不连续的元素/行/列

注意：

  1. 切片修改会改变原数组，如果需要独立副本，用 .copy()
  2. 索引越界会报错，切片越界不会报错（自动截断）
  3. 多维索引中，逗号分隔多维度，冒号代表当前维度全选

五、排序

NumPy 提供了高效、；灵活的数组排序方法，比 Python 原生列表排序更快

排序函数

  • np.sort() :返回排序后的新数组（不修改原数组）

该函数默认升序排序，会生成一个新的有序数组，原数组保持不变

  • np.argsort() :返回排序后的索引（不直接排序）

返回的是元素从小到大的位置索引，一般用于按条件排序或者关联数据

  • ndarry.sort() :原地排序（直接修改原数组）

直接对原数组排序，不生成新数组，节省内存

降序排序

由于在 NumPy 中没有直接的降序排序函数，可以用切片反转

按行/按列排序（二维数组）

通过 axis 参数指定排序方向

  • axis =0即按列排序
  • axis =1即按行排序

部分排序

通过函数 np.partition() 来实现，只找前 kth 小的元素，不完整排序，但是速度快

原理：

  1. 第 kth 小的元素别放置在他完全排序后应在的正确位置上
  2. 所有小于或等于第 kth 小的元素的值都会被移动到他的前面
  3. 所有大于或等于第 kth 小的元素的值都会被移动到他的后面

六、数组的形状操作

NumPy 数组形状操作就是改变数组的维度、尺寸、结构，不修改数据本身，是数据分析、机器学习必用技能

.shape 查看数组形状

.reshape() 改变形状

.reshape() 不改变数据，只改变结构

自动计算维度：用 -1

这个可以让 NumPy 自动计算行数/列数

展平数组（多维变一维）

  • 法1：.flatten() 这个会返回新的数组
  • 法2：.ravel() 这个会计量的返回视图，因此修改 .ravel() 返回的数组会影响原始数组

np.expand_dims() 增加维度

给数组加一个维度，常用于机器学习输入格式

np.squeeze() 压缩维度

去掉大小为1的维度

.T 数组转置（行列互换）

七、数组生成函数

从已有数据创建

np.array()

从列表、元组创建数组

np.asarray()

类似于 array 但不复制原数据，效率高

固定形状、固定值的数组

np.zeros(shape) 全零数组

np.ones(shape) 全1数组

np.full(shape,val) 指定值填充

np.eye(n)/np.identity(n) 单位矩阵（对角线1，其余都是0）

np.empty(shape)

这是一个空数组，只分配内存，并不会初始化，值随机

按范围、步长生成

np.arange(start,stop,step)

类似 Python 中的 range ，但返回数组，左闭右开，支持小数

np.linspace(start,stop,num)

这是一个等差数列，指定总个数

np.logspace(start,stop,num)

这是等比数列（10的幂）

随机数组

np.random.rand(*shape)

生成 0~1 均匀分布的数组，类似的还有 np.random.randint(low,high,size) 和 np.random.random() ,分别是随机生成整数数组和浮点数数组

np.random.seed()

这是一个随机种子，NumPy 生成的随机数是伪随机的，可以通过设置随机种子来保证每次生成的随机数相同，能保证别人运行自己的代码时得到相同的结果

网络坐标数组

np.meshgrid(x,y)

生成一个二维网络坐标（画图、矩阵运算常用）

np.diag()

取对角线 或 生成对角矩阵

八、文件读写

NumPy 中的文件读写主要有两大类：自己的 .npy / .npz 格式 和 文本文件（txt / csv）

文本文件读写（.txt / .csv）

  • 保存数组到文件： np.savetxt()

常用参数：

  1. delimiter=','：分隔符
  2. fmt='%.2f'：控制数字格式（保留两位小数等）
  3. header='col1,col2,col3'：加表头
  • 从文本读取数组： np.loadtxt()

NumPy 专用格式（.npy / .npz）

适合保存多维数组、大数据，读取速度快

.npy 单个数组

.npz 多个数组
